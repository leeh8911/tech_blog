# Git
## Git commit 날짜 바꾸기
* `git commit --amend --no-edit --date "DD MM YYYY hh:mm:ss"`
* 특정 커밋에 대해 `--amend`는 이전 커밋 내용에 덮어쓰기의 의미
* `--no-edit`은 현재 커밋 내용을 수정사항 없이 반영하겠다는 의미
* `--date`는 날짜에 대한 옵션
